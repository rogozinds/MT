Testing is an important part of software development process.
To test easily a huge amount of code automated testing is needed. 
This work presents Vaadin Testbench - a testing tool for Vaadin applications that
 helps to write tests which simulate user actions on a Web page. It starts from
 describing architecture of a Vaadin framework, focusing on essential parts for creating
 the Vaadin testing tool. The comparison with other testing tools, such as Selenium,
 shows the importance of creating Vaadin Testbench.
  Finally, the paper describes Vaadin Testbench 4.0.0 development process and shows
   advantages of its usage for testing Vaadin applications.
